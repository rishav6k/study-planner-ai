# 📚 AI Study Planner

A smart study planner web app built using Gradio.

## 🚀 Features
- Generates weekly study plan
- Supports multiple subjects (Math, Science, Hindi, Biology, etc.)
- Handles weak topics with structured phases:
  - Concept
  - Practice
  - Revision
  - Test
- Realistic daily hour distribution
- Download plan as PDF

## 🧠 How it works
Enter:
- Subject
- Study hours per day
- Number of days
- Weak topic (optional)

The app generates a balanced and structured study plan.

## 🛠 Tech Stack
- Python
- Gradio
- FPDF

## 📌 Usage
Click on "Generate Plan" after filling inputs to get your study plan and PDF report.
